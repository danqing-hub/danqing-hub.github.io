为Hexo博客更换主题

1.把目录定位到blog（你的博客存放的根目录）
2.下载主题（用git克隆）

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

3.修改

vim _config.yml

把theme:landscape修改成更换的主题：yilia

4.重新情理

hexo clean

5.重新生成

hexo g

6.重新部署一下

hexo s

可以看到主题已经发生了变化

7.推到github上

hexo d

等待更新完成后，可以看到和自己本机的预览效果相同
