博客部署到github

登录到github

**然后点击下方的create完成创建

安装git，用于上传文件到github

在终端输入

cnpm install --save hexo-deployer-git

编辑_config.yml文件

vim _config.yml

在type后面添加

新建的github的地址如下：

输入完成后–》按Esc–》:wq 即可保存退出

配置git
  • 写入你的邮箱和用户名即可
部署到远端

hexo d

会提示你输入github的用户名和密码，根据要求输入即可

完成后可以看到

这样我们的部署工作就做好了

打开新建的github仓库，刷新

可以看到生成了很多的文件

可以看到，博客在github上部署成功了！！！
